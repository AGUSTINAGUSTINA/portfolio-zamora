App conceptual de seguridad

Wipeout es una app conceptual diseñada para actuar en situaciones crí­ticas como el robo o pérdida de un dispositivo móvil. Permite cerrar sesiones activas previamente configuradas y proteger información personal de forma inmediata.

Su propuesta se centra en la velocidad, la simplicidad y la toma de decisiones claras en contextos de emergencia.

Usuario Principal

Flujo pensado para la persona dueña del dispositivo: acceso rápido y claro para iniciar el proceso de protección, con mensajes simples y jerarquía visual para reducir fricción.

El usuario principal crea una cuenta y configura previamente qué aplicaciones, cuentas o credenciales desea proteger.

En caso de emergencia, esta configuración permite ejecutar acciones críticas de forma rápida desde otro dispositivo, minimizando el riesgo de acceso no autorizado.

Usuario Invitado

Modo pensado para quienes usan el equipo de forma temporal: usuario invitado, accediendo desde un dispositivo externo, descargando la aplicación e iniciando sesión como usuario invitado.

A partir de allí, puede ejecutar el protocolo de emergencia, que incluye la eliminación de cuentas y datos sensibles previamente configurados.

El flujo prioriza la claridad y la rapidez al ejecutaracciones en un contexto de urgencia.

Decisiones de diseño

Debido al contexto de uso (situaciones de estrés y urgencia), se priorizaron:

  • Mensajes claros y directos para reducir la carga cognitiva
  • Acciones críticas con confirmación explícita
  • Jerarquía visual enfocada en la acción principal
  • Minimización de pasos para ejecutar el protocolo
